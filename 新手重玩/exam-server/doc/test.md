# 试题管理页面接口
## 添加试题页面接口

### 提交试卷接口

#### URL地址
```js
    /test/saveTest
```
#### 请求方式
```js
    post
```

#### 参数
```js
     content,//题目主题内容
     answer,//题目答案
     title,//题目标题
     timeId,//考试时间ID
     classId,//课程类型ID
     titleId,//题目类型ID
     uid,//用户ID
```
#### 返回值
```js
     code,//0代表失败 1代表成功
     msg,//返回结果描述
```
